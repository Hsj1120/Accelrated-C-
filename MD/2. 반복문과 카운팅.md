## 2.3 주어진 개수만큼 행 출력하기

인사말과 위 아래 테두리는 각각 한 행을 차지하므로 행의 개수는 총 3개이다.

인사말과 테두리 위 아래 사이에 남겨 둘 공백 문장의 2배에 방금 언급한 3개의 행을 더한다면 출력할 행의 전체 개수를 구할 수 있다.

```c++
// 인사말과 한쪽 테두리 사이의 공백 개수
const int pad = 1;

// 출력할 행의 전체 개수
const int rows = pad * 2 + 3;
```

pad는 인사말과 한 쪽 테두리 사이의 공백 개수를 저장하는 상수

pad를 사용하여 출력하고자 하는 행의 전체 개수를 제어하는 상수 rows 값을 계산한다.



다음 단계로 넘어가보면 

```c++
	// 입력과 출력을 구분
	std::cout << std::endl;

	// rows만큼의 행을 출력
	int r = 0;

	// 불면성 : 지금까지 r개 행을 출력
	while (r != rows)
	{
		// 하나의 행을 출력(2,4에 설명)
		std::cout << std::endl;
		r++;
	}
```

while문을 사용하여 하나씩 한 행을 출력할 수 있도록 한다.



### 2.3.1 while 문

while문은 주어진 조건이 참일 때 실행문을 반복 실행한다.

while 문의 형식은

```
while(조건)
	실행문
```

while문은 조건을 판별하는 것으로 시작하며 '실행문'을 **본문**이라고도 한다.

조건이 거짓이면 본문을 실행하지 않고, 조건이 참이면 본문을 한번 실행한 수 다시 조건을 판별한다.

while문은 조건이 거짓이 될 때까지 조건 판별 및 본문 실행을 반복한다.

예제 프로그램의 while문은 r값이 rows와 같지 않는 동안 중괄호 안에 있는 모든 내용을 실행하라는 의미이다.

프로그램 가독성을 위해 들여쓰기를 습관화 하자.



위의 코드에서 실행문 뒤에 세미콜론이 없는데 그 이유는

실행문은 실제로 실행문 하나이거나 하나의 **블록**(중괄호로 둘러싸인 0개 이상의 실행문)이다.

하나라면 세미콜론으로 끝날 것이므로 '실행문' 뒤에 세미콜론이 필요하지 않고,

하나의 블록이라면 } 기호로 끝날 것이기에  실행문 뒤에 세미클론이 필요하지 않다.



while문은 **조건**을 판별하는 것으로 시작하며 조건은 진릿값을 반환하는 표현식이다.

r != rows를 살펴보면 **비항등 연산자**인 != 를 사용하여 r과 rows를 비교한다.

표현식의 결과는 진리값을 나타내는 bool 타입이며 참 또는 거짓을 값으로 갖는다.



```c++
++r;
```

++는 **증가 연산자** 이고 변수 r을 1값만큼 증가시킨다. 다음처럼 작성할 수 있다

```c++
r = r + 1;
```



### 2.3.2 while문 설계하기

while문을 작성하고 이해하는데 유용한 두가지 개념이 있다.



첫번째는 while문이 종료할 때 조건은 반드시 거짓이어야 한다.

예를 들어 while문이 종료할 때는 r != rows가 거짓일 때, 즉 r이 rows와 같은 값일 때다.



두번째는 while문에서 항상 참인 **루프불변성**이다.

루프 불변성이란 **루프가 반복될 때마다 항상 유지되는 '진행상태의 약속'**이다

의도한대로 프로그램이 동작할 수 있게 불변성을 정해야 하고, 적절한 때 불변성이 참이 되도록 프로그램을 작성해야 한다.

불변성은 실제로 실행하는 코드가 아닌 주석 형태이다.



코드로 불변성에 대해 설명하자면

```c++
// r을 0으로 설정하면 불변성은 참
while (r != rows)
{
	// 이 지점에서 불변성은 참
	// 하나의 행을 출력하면 불변성은 거짓
	std::cout << std::endl;

	// r을 증가시키면 불변성은 다시 참
	r++;
} // 따라서 이 지점에서 불변성은 참	 		
```

이 코드에서 필요한 불변성은 **'지금까지 r개의 줄을 올바르게 출력했다'**라는 불변성을 가져야 한다.

두번의 불변성을 검증해야 하는데 

첫번째는 while문이 처음으로 조건을 판별하기 바로 전이다. 어떠한 행도 출력하지 않았으므로 r을 0으로 설정한 것으로 불변성을 만족시킨다.

두번째는 while문 조건 판별에서 참일 때마다 while문의 조건과 본문을 거쳐 본문 끝에 도달했을 때 불변성이 참인지를 검증해야 한다.

하나의 행을 출력하면 r은 출력한 행 개수와 달라지므로 불변성은 거짓이 된다. 출력된 행을 반영하고 r 값을 증가시키면 불변성은 다시 참이 된다.

두 가지 요구사항이 모두 참이므로 while문이 끝났을 때 r == rows라는 것도 알 수 있다.



## 2,4 행 출력하기

행을 출력하기에 처음으로 살펴볼 것은 출력한 모든 행이 '같은 길이'라는 것이다.

해당 개수는 인사말의 길이에 두번의 공백과 별표 2개를 더해서 계산할 수 있다.

```c++
const std::string::size_type cols = greeting.size() + pad * 2 + 2;
```

std::string::size를 살펴보면

첫번째 ::는 범위 연산자이고 std::string은 std라는 네임스페이스에 속한 string이라는 기능을 의미한다.

두번쨰 ::는 string이라는 클래스에 속한 size_type이라는 기능을 사용하겠다는 의미이다.

size_type은 std::string 타입의 문자 개수를 보유하는데 적합한 타입이며 문자열의 길이를 담는 지역 변수가 필요할 때 사용한다.

cols를 int 타입으로 지정할 수 있으나 cols의 값은 입력 내용의 길이에 따라 다르고 입력 내용의 길이를 제어할 수 없다.

만약 누군가 아주 긴 문자열을 입력해서 int 타입의 수를 초과한 문자를 담은 문자열을 만들 수 있기에 문제가 될 수 있다.

따라서 특정 구조의 길이를 저장하는 변수를 정의할 때는 라이브러리 목적에 맞게 정의된 타입을 사용하자

문자열의 길이는 음수가 될 수 없다. 그리서 std::string::size_type은 **부호 없는 타임(unsigned)** 타입이다.



얼마나 많은 문자를 출력할 지 알아냈다면 또 다른 while문을 사용하여 그것을 출력할 수 있다.

```c++
std::string::size_type c = 0;

// 불변성 : 현재 행에서 c개 문자를 출력
while (c != cols) {
	// 하나 이상의 문자를 출력
	// 불변성을 참으로 유지하려고 c 값을 변경
}
```

이 경우는 하나의 행을 출력하는 while문과는 다르게 '하나 이상의 문자를 출력'이라고 명시하였는데 한 문자씩 출력하면서 불변성을 검사할 필요가 없고 출력하는 문자의 전체 개수가 정확히 cols와 같은지만 확인하면 된다.



### 2.4.1 테두리 문자 출력하기

첫번째 행과 열, 마지막 행과 열은 별표가 출력되어야 하며  루프 불변성을 이용해서 별표의 출력 여부를 결정할 수 있다.

만약 r이 0이면 불변성 때문에 아직 출력한 행이 없다는 뜻이고 즉, 첫번째 행을 출력하는 중임을 의미한다.

또한 r이 rows-1이면 이미 마지막 행을 출력하는 중임을 의미한다.

이와 비슷하게 추론하면 c가 0이면 첫번째 열을 출력하는 중이고 c가 cols - 1이면 마지막 열을 출력하는 중이다.

이를 사용해서 코드를 작성하면

```c++
// 불변성 : 현재 행에서 c개 문자를 출력
while (c != cols) {
	if (r == 0 || r == rows - 1 || c == 0 || c == cols - 1) {
		std::cout << "*";
	}
	else {
		// 하나 이상의 테두리외 문자를 출력
		// 불변성을 참으로 유지하려고 c 값을 설정
	}
}
```

이렇게 작성할 수 있다



#### 2.4.1.1 if문

while문의 본문은 if문을 포함하는 하나의 블록으로 구성되며 별표를 출력할 지 결정하려고 if문을 사용한다

if문은 다음 두가지 형식 중 한 가지를 사용할 수 있다

```c++
if (조건)
    실행문
    
if (조건)
    실행문1
else
    실행문2
```

while문과 마찬가지로 '조건'은 참 또는 거짓을 반환하는 표현식이다.

'조건'이 참이면 프로그램은 if 이후의 실행문을 실행하고 두번째 형식에서는 '조건'이 거짓이면 else 이후의 실행문을 실행한다.



#### 2.4.4.2 논리 연산자

```c++
r == 0 || r == rows - 1 || c == 0 || c == cols - 1
```

이 조건은 r이 0 또는 rows-1이거나 c가 0 또는 cols -1이면 참이다.

== 연산자와 || 연산자를 사용하는데, C++ 프로그램은 == 기호를 사용해서 **항등**여부를 판별한다.

따라서 r == 0은 값이 0인지 아닌지 나타내는 bool 값을 반환한다.

**논리합 연산자**인 ||는 피연산자 중 하나가 참이면 참을 반환한다.



관계연산자는 산술연산자보다 **우선순위**가 낮다.

예를 들어  r == rows - 1은 (r == rows) -1이 아닌 r == (rows -1)과 같다.

산술연산자인 -는 관계연산자 ==보다 우선 순위가 높기 때문이다.

우선순위를 무시하려면 하위 표현식을 괄호로 묶어야 한다.

(r == rows) - 1 처럼 사용해야 하며 이 표현식은 r과 rows를 비교한 결과에 1을 뺀 것이므로 r이 rows와 같은지에 따라 0 또는 -1을 반환한다.



논리합 연산자는 피연산자 중 하나가 참인지 판별한다

```c++
조건1 || 조건 2
```

'조건 1'과 '조건 2'는 진릿값을 반환하는 조건 표현식이다. || 표현식은 bool 값을 반환하며 조건 중 하나가 참이면 참을 반환한다.

||는 왼쪽 우선 결합성이 있다.

중요한 다른 속성 중 하나가 있다면 ||의 왼쪽 피연산자가 참일 때 오른쪽 피연산자의 진리값을 전혀 구하지 않으며 이를 **간략 평가법**이라고 한다.



### 2.4.2 테두리 외 문자 출력하기

이전 코드에 else문에 해당하는 실행문을 작성해야 한다.

해당 실행문은 테두리에 속하지 않는 문자를 처리해야 하며 이 문자는 인사말의 일부이거나 공백이다.

인사말의 첫번째 문자를 출력하려면 현재 지점이 인사말의 첫번째 문자를 출력해야 하는 행과 열이라는 것만 알면 된다.

행 위치는 별표만으로 이루어진 첫 번째 행의 다음행 부터 pad만큼의 행을 더해서 찾을 수 있다.

열 위치는 별표를 출력하는 첫번째 열의 다음 열부터 pad만큼의 공백을 더해서 찾을 수 있다.

r이 pad+1이고 c가 pad + 1인 위치가 인사말의 첫 번째 문자를 출력할 위치이다.

즉, r과 c가 모두 pad + 1과 같은지 확인해야 하고, 그 결과에 따라 인사말을 출력하고 그렇지 않다면 공백을 출력한다.

```c++
if (r == pad + 1 && c == pad + 1) {
	std::cout << greeting;
	c += greeting.size() ;

}
else {
	std::cout << " ";
}
```

if문 조건에서 **논리곱 연산자**를 사용하고 두 조건이 모두 참일 때만 참을 반환한다.

왼쪽 우선 결합성이 있으며 간략평가법을 따른다.

조건의 판별 결과가 참이면 인사말을 출력하는데 출력하면 c 값이 더 이상 현재 행에 출력한 문자 개수가 맞지 않아 불변성에 오류가 생긴다.

그래서 c 값에 출력한 인사말의 문자 개수를 더하여 불변성을 다시 참으로 만든다.

c 값을 변경하는 푷ㄴ식에서는 **복합 할당 연산자**를 사용핬다

C += greeting.size()와 c = c + greeting,size()와 같은 의미이다.

### 2.5.2 while문 대신 for문 사용하기

프로그램의 가장 바깥쪽 부분을 보면 이런 형태이다.

```c++
int r = 0;
while(r != rows){
    // 값을 변경하지 않는 부분
    ++r;
}
```

for 문을 작성해서 더 간단하게 작성할 수 있다.

```c++
for (초기 실행문 조건 ; 표현식)
	실행문
```

앞 for문은 본문을 거칠 때마다 r이 0부터 rows - 1 까지 연속된 값을 같는다.

0은 범위의 시작이고 이러한 범위를 **반 개방 범위**라고 하며 [시작 값, 마지막 보다 하나 큰 값) 이렇게 표기한다.



**for문**의 형태는 다음과 같다

```c++
for (초기 실행문 조건 ; 표현식)
	실행문
```

첫번째 행은 **for 헤더**이다 헤더는 실행문으로 이루어진 **for 본문**을 제어한다.

for 헤더의 초기 실행문은 for문이 시작할 때 한번만 실행한다. 초기 실행문에서 정의한 변수는 for문 밖에서 소멸한다.

for문을 반복실행하면서 조건도 반복해서 판별하고 조건이 참을 반환하면 for문에 본문을 실행한다.



### 2.5.3 중복 연산 줄이기

현재 코드에서 인사말을 출력하는 부분을 첫 번째 순서로 둔다면 다른 두 부분은 불변성을 유지할 수 있도록 c값을 증가시키는 코드를 하나로 줄일 수 있다.

```c++
if (인사말을 출력하는 위치){
    cout << greeting;
    c += greeting,size()
} else {
    if (테두리를 출력하는 위치)
        cout << "*";
    else
        cout << " ";
    ++c;
}
```

따라서 std::cin >> name을 실행하면 표준 입력에서 문자열을 읽고 문자열을 구성하는 문자들을 name에 저장한다.



### 2.5.4 완성된 프로그램

입력동작에서 부수적으로 발생하는 결과로 사용자의 이름을 묻는 메세지가 출력장치에 표시된다.

일반적으로 입출력 라이브러리는 출력작업의 최적화를 위해 **버퍼**라고 하는 내부 데이터 구조를 사용한다.

라이브러리는 버퍼를 사용하여 출력할 문자들을 모으고, 필요할 때만 버퍼의 저장한 내용을 출력하고 버퍼를 지운다.

이 방법으로 여러 번의 출력 작업을 효율적으로 처리한다.

```c++
#include <iostream>
#include <string>

// 표준 라이브러리에서 사용할 이름을 언급
using std::cout; using std::string;
using std::cin; using std::endl;

int main()
{	
	// 이름을 물음
	cout << "Please enter your first name : ";

	// 이름을 입력
	string name;
	cin >> name;

	// 출력하려는 메시지를 구성
	const string greeting = "Hello, " + name + "!";

	// 인사말과 한쪽 테두리 사이의 공백 개수
	const int pad = 1;

	// 출력할 행의 전체 개수
	const int rows = pad * 2 + 3;
	const string::size_type cols = greeting.size() + pad * 2 + 2;

	// 입력과 출력을 구분
	cout << endl;

	//rows개 행을 출력한다.
	// 불변성: 지금까지 r개 행을 출력
	for (int r = 0; r != rows; ++r)
	{
		string::size_type c = 0;
		// 불변성: 현재 행에서 c개의 문자를 출력
		while(c != cols)
		{
			if(r == pad + 1 && c == pad + 1)
			{
				cout << greeting;
				c += greeting.size();
			}
			else
			{
				if(r == 0 || r == rows - 1 || c == 0 || c == cols - 1)
					cout << "*";
				else
					cout << " ";
				++c;
			}
		}
		cout << endl;
	}
	// 다시 작성할 부분
	return 0;
}


```

## 2.7 핵심 정리

**연산자 우선 순위**

| 1순위 연산자 | 설명                        |
| ------------ | --------------------------- |
| x.y          | 객체 x의 멤버 y             |
| x[y]         | 배열/컨테이너 접근          |
| x++          | 후위 증가 (값 반환 후 증가) |
| x--          | 후위 감소 (값 반환 후 감소) |

| 2순위 연산자 | 설명                        |
| ------------ | --------------------------- |
| ++x          | 전위 증가 (증가 후 값 반환) |
| --x          | 전위 감소                   |
| !x           | 논리 부정                   |

| 3순위 연산자 | 설명   |
| ------------ | ------ |
| x * y        | 곱셈   |
| x / y        | 나눗셈 |
| x % y        | 나머지 |

| 4순위 연산자 | 설명 |
| ------------ | ---- |
| x + y        | 덧셈 |
| x - y        | 뺄셈 |

| 5순위 연산자 | 설명          |
| ------------ | ------------- |
| x << y       | 왼쪽 쉬프트   |
| x >> y       | 오른쪽 쉬프트 |

| 6순위 연산자 | 설명   |
| ------------ | ------ |
| x == y       | 같다   |
| x != y       | 다르다 |

| 7순위 연산자 | 설명     |
| ------------ | -------- |
| x && y       | 논리 AND |

| 8순위 연산자 | 설명    |
| ------------ | ------- |
| x \|\| y     | 논리 OR |



**타입**

- bool : 참 또는 거짓 중 한가지 진릿값 표현
- unsigned : 음수가 아닌 값만 표함하는 정수 타입
- short : 최소 16비트인 정수 타입
- long : 최소 32비트인 정수 타입
- size_t : 모든 객체 크기를 저장할 수 있는 부호 없는 정수 타입
- string::size_type : 모든 문자열의 크기를 저장할 수 있는 부호 없는 정수 타입



**반 개방 범위**

하나의 종점을 포함하며 두 종점을 모두 포함하지 않는다, [1,3)은 범위 1과 2를 포함하며 3은 포함하지 않는다.



**조건**

진리값을 반환하는 표현식, 조건에 사용된 산술값은 bool타입으로 변환되고, 0이 아닌 값은 참으로 변환되고 0은 거짓으로 변환
