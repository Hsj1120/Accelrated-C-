# 0.0 개요
```
// 간단한 C++ 프로그램
#include <iostream>

int main()
{
    std::cout << "Hello World!\n";
}
```

출력하면 다음과 같은 메세지가 보인다
 ```
 Hello, World!
 ```



# 0.1 주석

프로그램의 첫 행은 다음과 같다

```c++
// 간단한 C++ 프로그램
```
//** 는 한 행의 끝까지 이어지는 주석의 시작을 의미하며, 프로그램을 사람하게 설명하는 용도로 사용되며 컴파일러는 주석을 무시한다



# 0.2 #include
C++에서 여러 기본 기능은 **기본영역**이 아닌 **표준 라이브러리**에 속해 있다.

기본 영역에 속한 기능은 모든 C++에 사용할 수 있지만 표준 라이브러리 기능은 반드시 명시적으로 요청해야 사용할 수 있다.

표준 라이브러리 기능을 요청할 때는 #include 지시문을 사용한다

```
#include  <iostrem>
```
iostream은 순차적인 또는 연속적인 입출력을 지원하며 #include 지시문에서 iostream이 **꺽쇠괄호**로(<,>)로 묶여 있을 때는 **표준 헤더**라는 C++ 라이브러리 일부분을 참조한다.



# 0.3 main 함수
**함수**는 프로그램의 일부분에 이름을 붙인 것으로 프로그램의 다른 영역에서 **호출**하거나 실행할 수 있다.

모든 C++ 프로그램은 반드시 main 함수를 포함하며 C++ 구현체는 main함수를 호출해서 프로그램을 실행한다

```c++
int main()
```

여기서 int는 정수를 나타내려는 용도로 미리 정의된 이름이며, main 뒤의 괄호 안에는 매개변수가 들어간다.



# 0.4 중괄호

이어 **중괄호**로 묶인 **실행문**을 살펴보면

```c++
int main()
{ // 왼쪽 중괄호
  // 실행문
} // 오른쪽 중괄호
```

C++에서는 프로그램 실행 시 중괄호 사이의 내용을 하나의 단위로 취급한다.

왼쪽 중괄호는 실행문의 시작을 표시하고, 오른쪽 중괄호는 실행문의 끝을 표시한다.

즉, 중괄호는 그 사이에 있는 모든 실행문이 하나의 함수에 속해 있음을 나타낸다



# 0.5 표준 라이브러리를 사용한 출력

```c++
std::cout << "Hello World!\n" << std::endl;
```

이 실행문은 표준 라이브러리의 **출력연산자**인 <<를 사용하여 Hello, world를 출력하고, 이어서 std::endl의 값을 출력한다

std::로 시작하는 코드는 해당 이름이 std라는 **네임스페이스**에 속해 있음을 나타내며, 네임스페이스는 관련 있는 기능을 하나의 이름으로 모아놓은 것이다.

std는 표준 라이브러리가 정의한 모든 기능이 속한 네임스페이스이며, 예를 들어 iostream 표준 헤더는 cout과 endl을 정의하며 이를 참조하려면 std::cout과 std::endl로 표기해야 한다.



# 0.6 반환문

```c++
return 0;
```

반환문은 해당함수의 실행을 끝내고 함수를 호출한 프로그램에 return과 세미콜론 사이의 값을 반환한다.

반환 값은 함수가 반환할 것이라고 명시한 타입과 일치해야 한다.



# 0.7 심화

## 표현식

**표현식**은 프로그램이 무언가를 계산하도록 요청한다. 표현식을 실행할 때의 결과가 프로그램이나 구현체에 상태에 영향을 미친다

예를 들어 3 + 4는 결과로 7을 반환하는 부수적으로 발생하는 결과가 없는 표현식이며

```c++
    std::cout << "Hello World!\n" << std::endl;
```

위의 표현식은 부수적으로 발생하는 결과로 표준 출력 스트림에 Hello, world!를 넣고 현재 행을 마친다.



표현식은 연산자와 피연산자를 포함해 형태가 다양하다. 위의 코드에서 **<<** 기호는 연산자이고 std::cout, "Hello, world!", std::endl은 피연산자이다



## 타입

모든 피연산자에는 **타입**이 있으며 해당 데이터 구조에 적합한 연산이 무엇인지 나타내는 것이며, 연산자의 역할은 피연산자의 타입에 따라 달라진다.

각 타입에는 이름이 있으며 정수를 표현하는 타입의 이름은 int라고 정의한다.

std::cout은 std::ostream의 타입이다.



## 왼쪽 우선 결합성

예제에서는 2개의 피연산자가 존재하는데  2개의 <<연산자와 3개의 피연산자가 존재할 수 있는 이유는 << 연산자에 **왼쪽 우선 결합성**이 있기 때문이다.

하나의 표현식에 << 연산자가 2번 이상 나타날때 각 << 연산자는 자신을 기준으로 왼쪽 모든 부분과 오른쪽의 최소 부분을 피연산자로 사용한다.

첫번째 << 연산자의 오른쪽 피연산자는 "Hello, world!" 이고 왼쪽 피연산자는 std::cout이다.

두번째 << 연산자의 오른쪽 피연산자는 std::endl이고 왼쪽 피연산자는 std::cout << "Hello, world!"이다

왼쪽 우선 결합성에 따라 

```c++
(std::cout << "Hello, world!") << std::endl;
```

다음과 같다

순서로 따지면 

1.  첫번째 << 연산자에서 왼쪽 피연산자에 해당하는 스트림에 오른쪽 "Hello, world!"를 넣는다
2. 두번째 << 연산자의 왼쪽 피연산자는 std::ostream 타입인 std::cout을 반환하는 표현식이고 오른쪽 피연산자는 **조작어**인 std::end이 된다.
3. 스트림에 조작어를 넣으면 문자를 넣는 것과 다른 동작을 실행하는데, 오른쪽에 피연산자가 조작어라면 << 연산자는 주어진 스트림에 관한 조작어의 동작을 실행하고, 해당 스트림을 반환한다.
4. std::endl이 실행하는 동작은 현재 행에서 출력을 마치게 된다.



# 범위

**범위**는 해당 이름의 문맥적 의미가 유지되는 프로그램의 영역을 말하며 예제 코드에는 2가지가 있다.

첫 번째 범위는 네임스페이스이며 네임스페이스는 관련 있는 기능들을 모아놓은 것이다.

표준라이브러리는 std라는 네임스페이스에 속한 모든 기능을 미리 정의되어 있다.

예를 들어 std::cout은 std라는 네임스페이스의 정의된 기능인 cout임을 의미한다.

:: 연산자는 **범위연산자**로 ::연산자 왼쪽에는 범위의 이름이 있는데 범위의 이름은 std라는 네임스페이스, 오른쪽에는 왼쪽에 범위에서 정의한 이름이 있다.

따라서 std::cout은 '네임스페이스 std 범위에 있는 cout이라는 이름'을 의미한다.

두번째 범위는 **중괄호**이다. main함수를 포함한 모든 함수의 본문은 하나의 범위이다.



# 0.8 핵심 정리



## 프로그램 구조

C++ 프로그램은 일반적으로 **자유 형식** 이다. 즉 인접한 기호들이 함께 실행되는 것을 방지할 때만 공백이 필요하며, 특히 행 바꿈은 공백의 한 종류이며 특별한 의미를 가지지 않는다

자유 형식이 아닌 세가지 독립요소들은 다음과 같다

- 문자열 리터럴 : 큰따옴표로 묶인 문자들은 여러 행에 작성할 수 없다.
- #include : 하나의 행으로 명시해야 한다.
- 주석 : // 이후에 오는 모든 내용이 주석이며 현재 행의 끝에서 끝난다.



## 타입

데이터 구조 및 해당 데이터 구조의 연산을 정의한다.

C++에는 두가지 종류의 타입이 있는데 int처럼 프로그래밍 언어의 기본 영역 안에서 정의한 것과 std::ostream처럼 프로그래밍 언어의 기본 영역 밖에서 정의한 것이 있다.



## 네임스페이스

관련 있는 기능들을 모아두었으며 표준 라이브러리 기능들은 std라는 네임스페이스에 정의된다



## 문자열 리터럴

큰 따옴포(")에서 시작하고 끝난다. 긱 문자열 리터럴은 행 하나에 완전히 표시되어야 하며 문자열 리터럴 일부 문자는 앞에 역슬레시(\\)가 있을 때 특별한 의미를 가진다

- \n : 행바꿈
- \t : 탭 문자
- \b : 백스페이스 문자

- \\" : 큰따움표를 문자열의 경계가 아닌 문자열 일부로 취급
- \\' : 문자 리터럴의 일관성을 위해 문자열 리터럴에서 '와 같은 의미로 취급한다
- \\\ : \\를 문자열 일부로 취급하고 다음에 오는 문자를 일반 문자로 처리한다.



## 정의문과 헤더

C++ 프로그램이 사용하는 모든 기능은 저마다 정의문이 있다.

표준 라이브러리는 헤더에 이름들을 정의하며 프로그램은 #include 지시문으로 헤더에 접근한다.

그러므로 기능을 사용하기 전에 이름이 정의되어야 하며, #include 지시문은 헤더에 있는 기능을 사용하기 전에 먼저 등장해야 한다.



## main 함수

모든 C++ 프로그램은 int 타입을 반환하는 main함수를 반드시 하나만 정의해야 하며 구현체는 main 함수를 호출해서 프로그램을 실행한다.

main 함수에서 0을 반환하면 성공, 0이 아닌 값을 반환하면 실패를 나타낸다.



## 중괄호와 세미콜론

중괄호로 묶인 0개 이상의 연속된 실행문을 **블록**이라고 하며 각각 순서대로 실행된다.

함수는 본문이 단 1개의 실행문으로 구성되도라도 반드시 중괄호로 묶어야 한다.

세미콜론으로 끝나는 표현식은 **실행문**이라고 하며 부수적으로 발생하는 결과를 얻기 위해 실행하는 것이며 표현식이 반환하는 결과는 무시한다.



## 출력

std::cout << e를 실행하면 표준 출력 스트림에 e값ㅇㄹ 쓰고 연속적인 출력동작을 위해 ostream 타입인 std::cout을 반환한다.
